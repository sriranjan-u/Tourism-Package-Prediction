# Tourism Package Prediction - MLOps Project

This project predicts whether a customer will purchase a tourism package using machine learning and MLOps practices.

## Project Structure
- data/: Dataset storage
- src/: Model training scripts
- models/: Saved models
- app/: Streamlit deployment
- mlops/: CI/CD pipeline

## Tools Used
Python, Scikit-learn, MLflow, Streamlit, Docker, GitHub Actions, Hugging Face
